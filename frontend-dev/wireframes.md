# Wireframes for Researcher Interfaces

## 1. Dashboard Overview
- **Layout:**
  - **Header:** Application logo and title.
  - **Token Meter:** Positioned at the top right to indicate current token usage.
  - **Recent Activity:** Summary of user actions (e.g., recent searches, generated reports) displayed below the token meter.

- **Key Components:**
  - Token usage graph (line or bar chart).
  - List of recent activities (with timestamps).

- **User Flows:**
  - Click on recent activities to view details.

---

## 2. Search Page
- **Layout:**
  - **Header:** Application logo and navigation to home/dashboard.
  - **Query Input:** Central large input box for user queries.
  - **Filters:** Below the query input for narrowing down searches (e.g., date range, keywords).
  - **Results List:** Below filters, displaying results genuinely based on input.

- **Key Components:**
  - Input field for search queries.
  - Filters (checkboxes, dropdowns).
  - Results displayed as cards or list items with titles, snippets, and links.

- **User Flows:**
  - Enter query and apply filters.
  - Click a result to view detailed information.

---

## 3. Private Corpus Manager
- **Layout:**
  - **Header:** Application logo and link to dashboard.
  - **Saved Papers List:** Table or card view displaying all saved papers.
  - **Notes Editor:** Below the list for editing notes related to a selected paper.
  - **Import Button:** Prominently located at the top or bottom of the papers list.

- **Key Components:**
  - List of saved papers with titles and snippets.
  - Editable text area for notes.
  - Button for importing papers from external sources.

- **User Flows:**
  - Select a paper to view and edit associated notes.
  - Click import to add new papers.

---

## 4. Output Gallery
- **Layout:**
  - **Header:** Application logo and link to dashboard.
  - **Generated Literature Reviews List:** Thumbnails or cards showing all generated outputs with titles and summaries.

- **Key Components:** 
  - List of generated literature reviews.
  - Options for exporting or sharing reviews.

- **User Flows:**
  - Click an output to view/edit or share it directly.
